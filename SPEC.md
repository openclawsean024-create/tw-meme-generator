## 1. 問題陳述

### 1.1 目標使用者

| 族群 | 規模 | 痛點 |
|---|---|---|
| LINE 群組活躍使用者 | ~500 萬 | 想用梗圖回應但找不到對的、想自己加文字但不會 PS |
| FB 社團小編 | ~20 萬 | 經營社團需要每天產梗圖，現有工具需付費或操作複雜 |
| 年輕世代（16-30 歲） | ~300 萬 | 喜歡台灣在地梗（像極了愛情、我就爛、是在哈囉），但每次都要截圖再標文字 |

### 1.2 為什麼不做替代方案

- **Canva / Photoshop**：操作複雜、付費、中文字型不友善
- **既有的「梗圖產生器」網站**：版型老舊、無台灣梗、無 OCR 自動定位
- **自己用 LINE 截圖再編輯**：步驟多、無法精準定位
- **我們的解法**：雙模式編輯器（替換/新增文字）+ OCR 自動偵測位置 + 25 個預載台灣經典梗圖 + 一鍵下載 PNG，純前端零 API key

---

## 2. 解決方案

### 2.1 核心價值主張

> 「25 個台灣經典梗圖，文字雙模式編輯，30 秒產出可分享的 PNG。」

### 2.2 使用者流程

1. 從 25 個預載台灣梗圖中選擇（像極了愛情/我就爛/是在哈囉 等）
2. 選擇模式：**替換文字**（自動 OCR 偵測現有文字位置）或 **新增文字**（自選位置）
3. 編輯文字內容（支援中文 + emoji）
4. 預覽後下載 PNG
5. 分享到 LINE / FB / IG

## 3. 功能清單

### 3.1 MVP（必做）

- [ ] 25 個預載台灣經典梗圖（含像極了愛情/我就爛/是在哈囉/真的假的/傻眼貓咪 等）
- [ ] 雙模式編輯器：替換文字 + 新增文字
- [ ] OCR 自動偵測現有文字位置（Tesseract.js）
- [ ] 中文字型支援（內建 Web Fonts）
- [ ] 熱門排行（瀏覽次數 + 下載次數）
- [ ] 一鍵下載 PNG
- [ ] 社群分享按鈕（LINE / FB / IG link copy）

### 3.2 v2（加值）

- [ ] 上傳自製梗圖（使用者上傳 → 加入私房庫）
- [ ] AI 自動配文（輸入主題 → 自動生梗圖文字）
- [ ] 梗圖挑戰賽（限時主題投稿）
- [ ] 收藏 + 我的梗圖庫

### 3.3 明確不做

- 短影片梗圖（純圖片）
- 商用授權內容（不做商業版權梗圖）
- 多語系介面（先繁中）
- 與 IG API 即時整合（純 link copy）

## 4. 技術棧

| 層 | 選擇 | 理由 |
|---|---|---|
| 前端 | Next.js 14 + TypeScript | SSR 友善、SEO 強 |
| OCR | Tesseract.js (前端 WASM) | 純前端、零 API |
| 中文字型 | Google Fonts (Noto Sans TC) | 免費、繁中完整 |
| 圖片處理 | Canvas API | 純前端 PNG 產生 |
| 資料持久化 | localStorage（最愛梗圖） |
| 部署 | Vercel |

## 6. 風險與決策

| 風險 | 等級 | 緩解 |
|---|---|---|
| 梗圖版權爭議（原作者抗議） | 🟠 中 | 預載素材以「常見迷因 + 標註來源」；使用者自製不受限 |
| OCR 中文辨識率差 | 🟠 中 | 提供手動位置調整；Tesseract chi_tra 模型 |
| 伺服器流量（25 張圖） | 🟡 低 | 全部靜態檔 CDN 快取 |
| 不雅內容過濾 | 🟠 中 | 預載清單避開敏感內容；v2 加使用者回報機制 |
